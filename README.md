# Rocketnotes :rocket:

Este projeto fullstack combina os conceitos e funcionalidades desenvolvidos nos projetos anteriores, integrando uma API robusta e um front-end intuitivo. Combinando Node.js, React, SQLite e outras tecnologias, a aplicação foi criada para oferecer uma experiência completa ao usuário.

---

## 💻 Projeto

Este repositório contém uma aplicação completa, incluindo uma API (back-end) e uma interface web (front-end). O projeto tem como objetivo principal organizar notas e tarefas, oferecendo funcionalidades para criação, edição e exclusão de notas com praticidade.

---

## 🚀 Tecnologias Utilizadas

**Back-end:**
- Node.js
- Express
- SQLite
- Knex.js
- bcrypt.js
- JWT (JSON Web Token)
- Multer
- Cors

**Front-end:**
- ReactJS
- Vite
- React Router DOM
- Styled Components
- Axios
- React Icons

**Ferramentas de Desenvolvimento:**
- Nodemon
- Biome.js
- Eslint

---

## 🔖 Funcionalidades

**Front-end:**
- Interface amigável e responsiva
- Navegação dinâmica utilizando React Router DOM
- Formulários para cadastro, edição e exclusão de notas
- Armazenamento e exibição de dados consumidos via API

**Back-end:**
- Sistema de autenticação com tokens JWT
- Criação, leitura, atualização e exclusão de notas, tags e links
- Gerenciamento de usuários com senhas criptografadas
- Upload de arquivos utilizando Multer
- Suporte a CORS para integração com o front-end

---

## 💾 Configuração e Uso

Siga estas etapas para configurar o projeto localmente:

### Back-end:
1. **Clone o repositório**:
    ```bash
    git clone https://github.com/nop-dev/RocketNotes
    cd api
    ```
2. **Instale as dependências**:
    ```bash
    npm install
    ```
3. **Execute as migrações**:
    ```bash
    npm run migrate
    ```
4. **Inicie o servidor**:
    ```bash
    npm run dev
    ```
5. **Acesse a API**: `http://localhost:3333`

### Front-end:
1. **Navegue até o diretório do front-end**:
    ```bash
    cd web
    ```
2. **Instale as dependências**:
    ```bash
    npm install
    ```
3. **Inicie o servidor de desenvolvimento**:
    ```bash
    npm run dev
    ```
4. **Acesse a aplicação**: `http://localhost:3000`
